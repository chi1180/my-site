Youtube の登録チャンネルの新規投稿をメールで知る！

始めに

今回は、「Youtube の登録チャンネルの新規投稿をメールで知る！」ということで、Google App Script を使用して、そんな感じの奴を作成しました。
一応、新しい投稿があったら通知が来る～みたいなシステムはありますが、Chrome を起動しないと通知が来ないですし（私はメインブラウザを Firefox にしています）何かと一元管理がしたいわたくしですので、メールでその日の新規投稿を確認できるようにしてやろうと思いましたのです！
今回作成したものによって、登録チャンネルの新規動画を確認するつもりが別のどうでもいい（でも面白いんだよ）動画によって時間を失うということを回避できるのです。
うぇ～い♪

機能

題名のとおりですね。

具体的には、以下のような感じの処理になっています：

  1. youtube data api v3 を使用して、登録チャンネルの一覧を取得
  2. 1 をもとに、チャンネルID を使用して、そのチャンネルのその日に投稿された動画を取得
  3. 適切なメール文にフォーマット
  4. メールを送信してあげる

最終的には、定期実行さるために作成したスクリプトにトリガーを指定します。
その日に投稿された動画を送るので、その日の最後らへん（12時近く）に実行するようになるわけです。
