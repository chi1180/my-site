使い方

実際のところ、非常に簡単に環境を用意することができます。

  1. Node.js をインストール
  2. ytdl-mp3 が存在する git repository より、npm を介してのインストールコマンドを参照

以上で、あなたの PC のコマンドプロンプトは 世界の Youtube から 動画音源をダウンロードするマジックぱわーを手に入れるのでした…
・・・具体的に説明すると、
Node.js 入れる
コマンドプロンプトに以下のコマンド入力してエンター


npm install -g ytdl-mp3

これで後は動画の音源を入れる方法ですが…
以下を参照してコマンドプロンプトに入力して Enter すれば OK です。

Usage: ytdl-mp3 [options] <url>

A NodeJS package and command-line tool for downloading music from YouTube, including automatic retrieval of ID3 tags and album art via iTunes.

Arguments:
url url of video to download

Options:
-V, --version output the version number
-o --output-dir path to output directory
-n --no-get-tags skip extracting/applying id3 tags
-v --verify-tags verify id3 tags fetched from itunes
-h, --help display help for command

>>>ytdl-mp3 https://www.youtube.com/watch?v=uRE8oEP9_bI
