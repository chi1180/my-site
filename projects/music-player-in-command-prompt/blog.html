レデイ後

How to command

1. windows key を click
2. cmd と入力
3. cd music と入力（お好きなミュージックフォルダへお飛びください）
4. start music.mp3 と入力（お好きなミュージックファイルを入力してください）
5. exit と入力（command prompt に用がなくなれば）

上記をまとめると、

$ cd [path/to/music_dir] && start [music_file] && exit

となります！
Very easy !!!
